# MP3 Album Merger List Generator

FFmpeg の `concat` 機能で MP3 を連結するための **リストファイル
(`list.dat`)** と、 各曲の **開始時間・再生時間をまとめた情報ファイル
(`info.txt`)** を生成する PHP CLI スクリプトです。

アルバムディレクトリ内の MP3 ファイルを読み取り、ディレクトリ順 →
ファイル順に並べて処理します。

------------------------------------------------------------------------

# Features

-   アルバムディレクトリを自動検出
-   MP3 ファイルをソートして連結順を決定
-   `ffprobe` を使用して再生時間を取得
-   FFmpeg concat 用 `list.dat` を生成
-   曲名・開始時間・再生時間を `info.txt` に出力

------------------------------------------------------------------------

# Directory Structure

このスクリプトは次のような構成を想定しています。

    music/
    ├─ Album1/
    │  ├─ 01 Song.mp3
    │  ├─ 02 Song.mp3
    │
    ├─ Album2/
    │  ├─ 01 Track.mp3
    │  ├─ 02 Track.mp3

------------------------------------------------------------------------

# Requirements

-   PHP 7.4 以上
-   FFmpeg
-   ffprobe

Windows の場合、`FF_PROBE_PATH` を ffprobe のパスに設定してください。

``` php
define('FF_PROBE_PATH',"D:\\ffmpeg\\bin\\ffprobe.exe");
```

------------------------------------------------------------------------

# Usage

## 1. リスト生成

    php make_list.php <music_directory>

例

    php make_list.php D:\music

引数を指定しない場合、スクリプトと同じディレクトリが対象になります。

------------------------------------------------------------------------

## 2. 生成されるファイル

### list.dat

FFmpeg concat 用のリスト

    file 'Album1/01 Song.mp3'
    file 'Album1/02 Song.mp3'
    file 'Album2/01 Track.mp3'

### info.txt

曲名と開始時間

    01 Song.mp3　【開始時間】00:00:00　(再生時間 03:12)
    02 Song.mp3　【開始時間】00:03:12　(再生時間 04:05)

------------------------------------------------------------------------

## 3. MP3 を連結

生成された `list.dat` を使って FFmpeg で連結します。

    ffmpeg -f concat -safe 0 -i list.dat -c copy merged.mp3

------------------------------------------------------------------------

# How It Works

1.  指定ディレクトリ内のアルバムディレクトリを取得
2.  各ディレクトリ内の `.mp3` ファイルを取得
3.  ソートして連結順を決定
4.  `ffprobe` で再生時間を取得
5.  `list.dat` と `info.txt` を生成
